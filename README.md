# Tangerine Marketplace

This is a little marketplace frontend webpage interacting with [dummyJSON](https://dummyjson.com/).

## To Run

```sh
npm i && npm start
```

## Packages Used

- immutable
- jquery
- lodash
- next
- react
- react-dom
- react-grid-layout
- react-infinite-scroll-component
- react-pro-sidebar
- react-responsive
- react-responsive-masonry
- react-toggle
- react-typed
- reactjs-popup
